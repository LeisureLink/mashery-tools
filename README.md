# Mashery Tools

> Helper library and tooling to interact with mashery.

## Install

```sh
npm install --save @leisurelink/mashery-tools
```



## Usage
```js
import MasheryTools from '@leisurelink/mashery-tools';

const tools = new MasheryTools('key', 'secret');
```
